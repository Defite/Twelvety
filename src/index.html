The Fastest Eleventy Starter Project

Twelvety is a pre-configured Eleventy starter project built to be fast (you'll find it hard to make it slow). It includes:

  • Component architecture
  • CSS pipeline using Sass and CleanCSS
  • JS pipeline using Browserify, Babel and Uglify
  • Only includes CSS and JS used in each page
  • Critical CSS and defer of non-critical CSS
  • Minification of HTML, CSS and JS
  • Responsive picture shortcode (automatically applied to Markdown images)
  • Content hash of assets

Made and used by Greg Ives
